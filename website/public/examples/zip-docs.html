Overview

ZIP archive file format with local file headers, central directory, and end-of-central-directory record. Supports computed CRCs and file offsets.

Usage

To use the generated code, first generate encoders/decoders for your target language:

binschema generate --language ts --schema zip-archive-format.schema.json --out ./generated

Import and Use

import { CentralDirEntryEncoder, CentralDirEntryDecoder } from './generated/zip-archive-format.js';

// Encoding (object → bytes)
const encoder = new CentralDirEntryEncoder();
const bytes = encoder.encode({
  "version_made_by": 0,
  "version_needed": 0,
  "flags": 0,
  "compression_method": 0,
  "file_mod_time": 0,
  "file_mod_date": 0,
  "len_extra": 0,
  "len_comment": 0,
  "disk_number_start": 0,
  "int_file_attr": 0,
  "ext_file_attr": 0,
  "file_name": "example"
});

// Decoding (bytes → object)
const decoder = new CentralDirEntryDecoder(bytes);
const decoded = decoder.decode();
console.log(decoded);
